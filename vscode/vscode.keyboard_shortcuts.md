## Standard

- Navigation 
	- See the tree           : `Command + Shift + E` 
	- Move around the tree   : `Arrows` 
	- Open a file from tree  : `Command + (Down Arrow)`
	- Source Control 		 : `Command + Shift + R`
	- See the search sidebar : `Command + Shift + F` 
	- Command pane           : `Command + Shift + P`
	- Jump to editor         : `Command + 1`

- Settings                   : `Command + ,`
- Open file                  : `Command + P`
- New file                   : `Command + N`

- Increase font size		 : `Command + Option + -`
- Decrease font size		 : `Command + Option + +`

- Search the document        : `Command + F`
- Preview markdown           : `Command + Shift + V`
- Commenting the code        : `Command + /`
- Split editor               : `Command + \`



## Custom

- Hiding and unhiding 
	- Hide side-bar          : `Command + Shift + A`
	- Hide the bottom bar    : `Command + Shift + S`

- Folding 
	- Folding                : `Command + Option + [`
	- Unfolding              : `Command + Opttion + ]`
