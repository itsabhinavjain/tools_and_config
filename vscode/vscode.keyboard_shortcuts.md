## Standard

- Increasing and decreasing the font size 
- Navigation 
	- See the tree : `Command + Shift + E` 
	- Move around the tree :- `Arrows` 
	- See the search sidebar : `Command + Shift + F` 
	- Command pane : `Command + Shift + P`
- Open file : `Command + P`
- Search the document : `Command + F`
- Preview markdown : `Command + Shift + V`
- Commenting the code : `Command + /`
- Settings : `Command + ,`

## Custom

- Hiding and unhiding 
	- Hide side-bar : `Command + Shift + A`
	- Hide the below bar : `Command + Shift + S`

- Folding 
	- Folding - `Command + Option + [`
	- Unfolding - `Command + Opttion + ]`
